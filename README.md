# Biosfera

Клиентское веб‑приложение проекта «Биосфера» на **Next.js** (App Router). Репозиторий содержит UI, страницы и клиентскую логику.

## Технологии

- **Next.js** 16
- **React** 19
- **TypeScript**
- **Sass / SCSS Modules**
- **Zustand** (стейт)
- **React Hook Form** (формы)
- **Swiper** (слайдеры)
- **PhotoSwipe** (галерея/лайтбокс)
- **Nodemailer** (почтовые отправки)
- **Biome** (линт/форматирование)

## Скрипты

```bash
# дев-режим
npm run dev

# сборка
npm run build

# запуск собранного проекта
npm run start

# проверка качества кода
npm run lint

# автоформатирование
npm run format
```

## Особенности

- **App Router** и файловая маршрутизация страниц
- **Компонентная архитектура** (переиспользуемые блоки/секции)
- **SCSS Modules** для изоляции стилей компонентов
- **Формы** с валидацией на базе React Hook Form
- **Глобальный стейт** через Zustand
- **Интерактив**: слайдеры (Swiper) и галереи (PhotoSwipe)
